=head1 NAME

Hotline::FileListItem - File object used internally by Hotline::Client

=head1 SYNOPSIS

    use Hotline::FileListItem;

    $file = new Hotline::FileListItem;

    $file->name("foo.txt");
    $file->type("TEXT");
    ...

=head1 DESCRIPTION

Hotline::FileListItem is a simple class for storing and retrieving file
information,  You should never have to create your own
Hotline::FileListItem objects when using Hotline::Client.  Getting and (to
a lesser extent) setting attributes is all that should be necessary.

=head1 CONSTRUCTION

=over 4

=item new

Creates a new Hotline::FileListItem object with all attributes set to undef
(or zero for numeric attributes).

=back

=head1 METHODS

All the Hotline::FileListItem methods are simple attribute get/set routines.
If given an argument, they set an attribute.  In all cases, they
return the current value of the attribute.

=over 4
      
=item creator TEXT

The file creator, given as a four-letter Mac OS creator code ("TTXT", "SIT!",
etc.)

=item name TEXT

The file name.

=item size NUM

The size of the file in bytes.

=item type TEXT

The file type, given as a four-letter Mac OS type code ("TEXT", "PICT", etc.)
or Hotline::Constants::HTLC_FOLDER_TYPE for folders.

=back
